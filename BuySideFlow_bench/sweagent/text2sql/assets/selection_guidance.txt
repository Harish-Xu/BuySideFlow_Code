## secumain SECUCATEGORY values
- SECUCATEGORY=1: stocks (A-share, B-share)
- SECUCATEGORY=3: funds (ETF, LOF, closed-end fund)
- SECUCATEGORY=4: indices (e.g. 沪深300)
Use these when filtering secumain for the right security type.

## Index INNERCODE lookup (CRITICAL)

To find any stock index INNERCODE, ALWAYS query secumain -- never lc_indexbasicinfo:
  SELECT INNERCODE FROM secumain WHERE SECUCODE='000300' AND SECUMARKET=83 LIMIT 1   -- 沪深300
  SELECT INNERCODE FROM secumain WHERE SECUABBR='中证白酒' LIMIT 1                   -- named index

Then:
- Use that INNERCODE as INDEXINNERCODE in lc_indexcomponent to get constituent stocks (SECUINNERCODE)
- Use that INNERCODE directly in qt_indexquote for index price data
- lc_indexbasicinfo uses different key columns -- querying it for INNERCODE will cause errors

## lc_maindatanew vs lc_mainindexnew

| Use case | Table | Notes |
|----------|-------|-------|
| Pre-computed ratios (growth rates, ROE, AVGNPYOYPASTFIVEYEAR) | lc_mainindexnew | fast, ready-to-use |
| Raw financial items, consolidated statements | lc_maindatanew | filter MARK=1 for consolidated |
| IC analysis over multiple report periods (ENDDATE groups) | lc_maindatanew with MARK=1 | per-period granularity |
| INFOPUBLDATE (announcement date) | both tables | available in both |

## Column name rules
- secumain.SECUABBR: the ONLY valid column for stock display names. CHINAME and CHINAMEABBR do NOT exist.
- For point-in-time ST/*ST exclusion, use cs_riskalert with IMPLEMENTDATE/REMOVEDATE; do not rely only on the current abbreviation in secumain.SECUABBR.
- TRADINGDAY: date column in quote/valuation tables (qt_dailyquote, qt_indexquote, qt_fundsperformancehis, lc_dindicesforvaluation).
- TRADINGDATE: date column in qt_tradingdaynew (different name -- do not mix up).
- ENDDATE: financial reporting period end date (lc_mainindexnew, lc_maindatanew).
- INFOPUBLDATE: announcement/publish date (lc_mainindexnew, lc_maindatanew).
- AVGNPYOYPASTFIVEYEAR is a real column in lc_mainindexnew. Do not substitute any other column for it.
