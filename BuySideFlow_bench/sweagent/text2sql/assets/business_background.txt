## Return and factor formulas
- Log-price OLS slope annualization: exp(slope * 250) - 1.
- Compound daily return annualization: (1 + daily_return)^250 - 1.
- Period return annualization: (1 + total_return)^(250/n_trading_days) - 1.
- Spearman IC: rank-based correlation; do not replace with Pearson unless requested.
- Momentum: product(1 + r_t) - 1 using adjusted close/复权净值 over the requested window.
- Reversal: negative z-score of long-window momentum in the same cross-section.
- RSRS: slope or slope/se(slope) from high-price on low-price regression over N days; follow the requested variant.
- Tracking error: annualized std of portfolio return minus benchmark return.
- Information ratio: mean excess return divided by tracking error.
- Sharpe ratio: excess return divided by return std; use risk-free rate only when provided or clearly inferable.
- Calmar ratio: annualized return divided by absolute max drawdown.
